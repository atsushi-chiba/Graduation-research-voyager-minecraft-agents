# MineColonies 自律コロニー実験

Minecraft (Forge 1.20.1) と MineColonies mod を使い、**LLM (大規模言語モデル) が「市長」として自律的にコロニーを運営する**社会シミュレーション研究のwikiです。プレイヤーが街を作るのではなく、LLMが建物の配置・住民の職業割り当て・生産計画を判断し、市民 NPC がそれに従って生活する — そのふるまいを長時間観察して、AI 統治のダイナミクスや失敗パターンを分析することを目指します。

このwikiは高専卒業研究の成果と進行状況を、研究室のメンバー・後任者・興味を持ってくれた方に向けて公開するものです。

## 研究の3つの目標

- **① 2コロニー間の戦争** — 独立した2つの LLM 市長がそれぞれのコロニーを率い、資源競合や領土紛争を経て衝突するまでのプロセスを観測する
- **② 狭小空間での効率的発展** — 限られた領土内で市長がどのように優先順位を付け、資源を配分し、都市機能を最適化するかを観察する
- **③ 市民 NPC の SNS 発言** — OASIS 等のプラットフォームと連携し、市民個々の "声" (不満・喜び・思想) をソーシャルグラフ上に流し込む

## 現在の状態 (2026-07)

通常ワールド (起伏あり) で LLM 市長 (`council.js`) と自動供給デーモン (`supply_bot.js`) を組み合わせ、Forge mod "VoyagerBridge" を経由して自律コロニーが継続稼働中です。旧スーパーフラットワールドは検証用として保存されています。詳細は [システム構成](system.md) と [引き継ぎ](handoff.md) を参照してください。
